Question-1: What are the different ways to select an element in the DOM?

There are 5 ways in which you can select elements in a DOM using selectors. 1.getElementsByTagName() 2.getElementsByClassName() 3.getElementById() 4.querySelector() 5.querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML is used to get or set HTML content inside an element. innerText gets or sets only the visible text, ignoring hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where a parent element handles events for its child elements instead of attaching event listeners to each child separately.
